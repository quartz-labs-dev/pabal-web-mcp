# Content 툴

콘텐츠 생성 툴.

## 툴 목록

### create-blog-html

콘텐츠로부터 블로그 HTML 문서 생성.

**입력:**
- `slug` (필수): 제품 슬러그
- `title` (필수): 블로그 포스트 제목
- `content` (필수): 블로그 콘텐츠 (마크다운 또는 HTML)
- `locale` (선택): 대상 로케일
- `author` (선택): 작성자 이름

**출력:**
- 생성된 HTML 파일 경로
- 블로그 메타데이터 (제목, 슬러그, 날짜)
- SEO 최적화된 구조

**기능:**
- 제목으로부터 자동 슬러그 생성
- 다중 로케일 지원
- SEO 메타데이터 통합
- 반응형 HTML 출력
